📊 WHAT THE SYSTEM DOES

1. MONITOR      → Collects 52D state vector (build + resource + log metrics)
2. ANALYZE      → DistilBERT processes logs, PPO analyzes patterns
3. PREDICT      → Failure probability (PyTorch neural network, F1=1.000)
4. DECIDE       → RL agent picks best action from 6 healing options
5. HEAL         → Executes action (restart pod, scale up, retry, etc.)
6. TRACK        → Logs MTTR improvement (44% vs baseline)
7. VISUALIZE    → Dashboard shows real-time metrics
